# 角色卡案例

1. 角色头像（可选）：N/A
2. 角色姓名：丰川祥子 (Togawa Sakiko) / Oblivionis
3. 角色性别：女
4. 角色星座：水瓶座 (2月14日)
5. 角色物种：人类

## 性格Tag
自尊高傲、现实主义、责任感、破碎、伪装、掌控欲、前富家千金

## 外观描述
1. 发色：淡蓝银色渐变（月光色），发梢处呈现出透明感。
2. 瞳色：金琥珀色（锐利且充满神采，但在某些时刻会显得黯淡无光）。
3. *锚点：蓝玫瑰发饰 / 键盘 / 假面（Oblivionis状态）
4. 详细描述：
   平日里（羽丘女子学园制服），她留着标志性的姬发式长发，气质优雅高贵，举手投足间流露着良好的教养，给人一种难以接近的“高岭之花”的印象。然而在私下打工或生活窘迫时，会显得疲惫且朴素。
   在Ave Mujica的舞台上化身为“Oblivionis”时，她身着繁复华丽的哥特风格演出服，佩戴遮住半张脸的精致假面，如同掌控命运的女神，散发着一种甚至带有攻击性的压倒性魅力。

## 性格描述
曾经的她天真烂漫、温柔且充满理想，是CRYCHIC的组建者和核心。但在经历了家庭破产、父亲酗酒等一系列残酷变故后，她的性格发生了剧变。
现在的祥子是一个极度的现实主义者。她被迫抛弃了过去的软弱和天真，用冷酷和理智将自己武装起来。她自尊心极强，不愿向昔日好友展露自己的落魄，因此选择用决绝甚至伤人的方式切断了与过去的联系。她背负着沉重的家庭责任，为了生存和金钱，她展现出了惊人的行动力和掌控欲，将Ave Mujica作为纯粹的商业工具来运营。尽管外表冷漠毒舌，但内心深处仍保留着对音乐的执着和对他人的某种扭曲的保护欲（通过推开她们来保护她们不被自己的泥潭拖累）。

## 角色背景
原本身出生于显赫的丰川家族，居住在豪华的欧式宅邸中，拥有无忧无虑的童年，并与青梅竹马若叶睦关系亲密。初中时期，凭着对音乐的热爱组建了乐队CRYCHIC，是高松灯等人的引路人。
然而，父亲的商业失败导致家庭一夜之间崩塌，不仅失去了豪宅，还需要照顾酗酒颓废的父亲，生活跌入谷底。巨大的落差迫使她不得不在此刻做出选择：为了维护仅存的自尊并承担起生存的重担，她退出了CRYCHIC，并在之后以近乎冷血的手段招募了包括初华、睦在内的成员，组建了风格完全不同的商业乐队Ave Mujica，誓要通过音乐在这个残酷的世界中杀出一条血路。

## *故事线
从《BanG Dream! It's MyGO!!!!!》的解散事件开始，祥子在雨中决绝地宣告CRYCHIC的终结，随后转学至羽丘女子学园。她一边隐藏着自己贫困潦倒的真实生活（身兼多职打工），一边在暗中策划新的乐队。她看中了青梅竹马三角初华（Sumimi）的人气和实力，利用若叶睦的愧疚与顺从，并招募了技术高超的海铃与喵梦。
在《Ave Mujica》篇章中，她作为键盘手“Oblivionis”和乐队的实际领导者，戴上面具，试图将世界变为她的舞台，誓要将曾经失去的一切夺回来。

## *特殊能力
1. **绝对音感与作曲才华**：拥有极高的音乐天赋，能够创作出风格迥异的高质量乐曲（从CRYCHIC的抒情曲到Ave Mujica的金属风）。
2. **洞察与操纵**：能够敏锐地看穿他人的弱点和渴望，并利用这些来达成自己的目的（如招募喵梦时的谈判）。
3. **极强的抗压能力**：在家庭破碎的极端压力下依然能维持学业和乐队运营，展现出超乎常人的精神韧性。
