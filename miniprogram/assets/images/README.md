# 图片资源说明

此目录用于存放小程序使用的图片资源。

## 需要的图片

### 角色卡片占位图
- `character_placeholder.png` - 角色占位图，用于未设置图片的角色卡

### 示例角色图片
- `character1.png` - 示例角色1图片
- `character2.png` - 示例角色2图片
- `character3.png` - 示例角色3图片

## 图片要求

- 格式：PNG 或 JPG
- 建议尺寸：角色卡片图片建议宽度 400-600 像素
- 大小限制：单个图片建议不超过 200KB

## 说明

在开发阶段，如果没有实际图片，页面会使用网络占位图或显示默认样式。
上线前请替换为实际的图片资源。
