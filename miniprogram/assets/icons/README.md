# TabBar 图标说明

由于微信小程序 TabBar 需要使用 PNG 格式的图标，请将以下图标放置到此目录：

## 需要的图标文件

1. `home.png` - 首页图标（未选中状态），建议尺寸 81x81 像素，灰色 #999999
2. `home-active.png` - 首页图标（选中状态），建议尺寸 81x81 像素，蓝色 #0052d9
3. `profile.png` - 我的图标（未选中状态），建议尺寸 81x81 像素，灰色 #999999
4. `profile-active.png` - 我的图标（选中状态），建议尺寸 81x81 像素，蓝色 #0052d9

## 图标要求

- 格式：PNG
- 尺寸：建议 81x81 像素
- 大小限制：每个图标不超过 40KB
- 建议使用简洁的线性图标风格

## 临时解决方案

如果暂时没有图标，可以从以下网站下载免费图标：
- https://iconfont.cn/
- https://www.flaticon.com/
- https://icons8.com/

或者使用 TDesign 图标库生成 PNG 图标。
